# Retail Sales Exploratory Data Analysis

## Project Overview
This project explores retail sales data to identify trends, seasonal patterns, and product performance using Python.

## Tools Used
- Python
- Pandas
- Matplotlib
- Google Colab

## Key Insights
- Sales exhibit clear seasonal trends
- A small number of products contribute the majority of revenue
- Product performance varies over time

## Recommendations
- Prioritize high-performing products
- Plan inventory around peak seasons
- Improve data collection for future analysis
